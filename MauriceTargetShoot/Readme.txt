Controls:
wasd = movements
enter = focus on open chat, send messages when in chat and stop focus on chat when outside
left mouse button = fire
r = reload
t = taunt
e = interact, only ammo and weapons. Doors open when in range

Door:
You can't walk through and open door if it's not yours, this to prevent players getting stuck in the rooms

Chat commands:
/global send to global
/advice send to advice
/trash send to trash-talk
/pm username   send message to said username in a seperate draggable window
